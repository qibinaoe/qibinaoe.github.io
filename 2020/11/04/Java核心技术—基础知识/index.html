零碎小知识

  • Java应用程序中的全部内容都必须放在类中

  • 源代码的文件名必须与公共类的名字相同（区分大小写）

  • 运行已编译的程序时，JVM从指定类中的main方法的代码开始执行

  • 根据Java语言规范，main方法必须声明为public、静态的

  • Java中所有函数都是某个类的方法

  • Java中的main方法必须有一个外壳(shell)类

  • System.out.println("We"),System.out对象调用println方法

  • .用于调用方法object.method(parameters)

  • 注释3种形式

    • // hello
    • /* */
    • 自动生成文档,以/**开始，*/结束

  • Java是强类型语言，共有8种基本类型

    • 整型

      • int 4字节 -2 147 483 648 ~ 2 147 483 647(刚过20亿)
      • short 2字节 -32768 ~ 32767
      • long 8字节
      • byte 1字节 -128~127
      • 长整型数值有一个后缀L或l(如4000000000L)
      • 十六进制数值有一个前缀0x或0X(如0xCAFE)
      • 八进制有一个前缀0
      • Java7开始，前缀0b或0B表示二进制数，如(0b1001)是9
      • Java7开始，可以为数字字面量加下划线(1_000_000)
      • 注：Java没有任何无符号(unsigned)形式的int、long、short或byte
      • Byte.toUnsignedInt(b)

    • 浮点类型

      • float 4字节 有效位数为6~7位
      • double 8字节 有效位数位15位
      • float类型有一个后缀F或f
      • 没有后缀的浮点类型总是默认位double
      • double后缀D或d
      • 十六进制表示浮点数值0.125 = 2-3 = 0x1.0p-3。尾数采用十六进制、指数采用十进制、指数的基数是2

    • 字符类型char(用于表示Unicode编码的代码单元)

      • 2个字节
      • 十六进制表示\u0000~\uFFFF

  • 算术运算符

    • / 参与/的运算的两个操作数都是整数时，表示整数除法： 15/2 = 7
    • / 参与/的运算的操作数存在浮点数时，表示浮点除法： 15.0/2 = 7.5

    • 码点和代码单元

      • char数据类型给是一个采用UTF-16编码表示的Unicode码点的代码单元。

        • 码点是一个编码表中的某个字符对应的代码值
        • 每个字符(char)用16位表示代码单元
          • 注意：在C和C++中，windows的32位机和64位机下，char占1个字节，这跟Java不一样。Java的char类型占2个字节(16bit)

      • 因此每个码点可能由多个代码单元组成

      • length()方法返回的是采用UTF-16编码表示的给定字符串所需要的代码单元

      • codePointCount()返回实际长度，即码点数量

      • charAt(n)返回位置n的代码单元

问题回答

  1. 取余和取模区别，

    • 取余rem 取模mod

    • 被除数 ➗ 除数 ＝ 商

    • 取余运算在计算中，使得商值向0的方向取整(换句话是说向0的方向舍弃小数位)

    • 取模运算在计算中，使得商值向负无穷的方向取整(换句话是说向负无穷的方向舍弃小数位)

    • 结论

      • 当被除数和除数符号相同时(同正或同负)，取余和取模结果相同
      • 当被除数和除数符号相反时(一正一负)，取余和取模结果不同

    • 前提： 对于一个除法(a = bq + r),如果 a不能被b整除（a÷b)，那么能够得到两种解，一种是bq > a，使得r < 0。另一种是bq< a，使得r > 0。

    • 例子

      • mod(7,3) = 1, rem(7,3) = 1
      • mod(-7,-3) = -1, rem(7,3) = 1
      • mod(7,-3) = -2, rem(7,-3) = 1
        • 两种解: (-2,1) , (-3,-2)
      • mod(-7,3) = 2, rem(-7,3) = -1
        • 两种解: (-2,-1) , (-3,2)

    • Java中%是取余。Math.floorMod是取模
