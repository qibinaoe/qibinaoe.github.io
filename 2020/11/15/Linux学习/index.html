Linux学习笔记

Linux入门我通过学习Linux就该这么学这个网站进行学习。书上的老师推荐的是Redhat7.0。由于Centos下载方便，我下载了Centos7.0,下载地址阿里云镜像.

基础知识笔记

Linux系统的内核负责对硬件资源的分配、调度等管理任务。

  • 系统内核对计算机正常运行具有重要作用
  • 用户应该通过系统调用接口开发出的程序或服务来管理计算机

Linux系统关系

  • 程序或服务
    • 系统调用接口
      • 内核
        • 硬件

shell 壳或终端

shell是一个命令行工具，用户借助shell来与内核(硬件)进行通信。通过用户的命令去调用相应的程序服务去完成工作

许多主流默认使用的终端是Bash(Bourne-Again Shell)解释器

命令

Linux命令格式

命令名称 [命令参数] 命令对象

之间用空格隔开

命令对象

要处理的文件、目录、用户等资源

命令参数

  • 长格式
    • 使用完整选项名称
    • man --help
  • 短格式
    • 单个字母缩写
    • man -h

常用系统工作命令

man

man man

查看man命令自身的帮助信息

  • / 从上至下搜索某个关键字, /linux
  • ? 从下至上搜索某个关键字, ?linux
  • n 定位到下一个关键字
  • N 定位到上一个关键字

echo

echo命令用于在终端输出字符串或变量提取后的值

echo [字符串|$变量]

date

date命名用于显示及设置系统的时间或日期

date [选项][+指定的格式]

以+号开头的参数，可以按照指定格式输出系统的时间或日期

  • %% a literal % %的转义
  • %a locale’s abbreviated weekday name (e.g., Sun) 缩写的工作日名称
  • %A locale’s full weekday name (e.g., Sunday) 完整的工作日名称
  • %b locale’s abbreviated month name (e.g., Jan) 月份的缩写
  • %B locale’s full month name (e.g., January) 完整的月份名称
  • %c locale’s date and time (e.g., Thu Mar 3 23:05:25 2005)
  • %C century; like %Y, except omit last two digits (e.g., 20)
  • %d day of month (e.g., 01) 本月中的第几天
  • %D date; same as %m/%d/%y 输出月/日/年
  • %F full date; same as %Y-%m-%d 输出年-月-日
  • %h same as %b 缩写月份
  • %H hour (00..23) 小时 24小时制
  • %I hour (01..12) 小时 12小时制
  • %j day of year (001..366) 今年的第几天
  • %m month (01..12) 月份
  • %M minute (00..59) 分钟
  • %n a newline 换行符
  • %p locale’s equivalent of either AM or PM; blank if not known
  • %P like %p, but lower case
  • %r locale’s 12-hour clock time (e.g., 11:11:04 PM)
  • %R 24-hour hour and minute; same as %H:%M (e.g. 21:54)
  • %s seconds since 1970-01-01 00:00:00 UTC
  • %S second (00..60) 秒
  • %t a tab 跳格
  • %T time; same as %H:%M:%S (e.g., 21:55:48 PM)
  • %u day of week (1..7); 1 is Monday
  • %U week number of year, with Sunday as first day of week (00..53)
  • %w day of week (0..6); 0 is Sunday
  • %W week number of year, with Monday as first day of week (00..53)
  • %x locale’s date representation (e.g., 12/31/99)
  • %X locale’s time representation (e.g., 23:13:48)
  • %y last two digits of year (00..99) 简写年份
  • %Y year 完整年份

将系统时间修改成硬件的时间

reboot

重启系统，默认只有root管理员能够重启

poweroff

关闭系统，默认只有root管理员能关机

wget

在终端下载网络文件

wget [参数] 下载地址

  • -b 后台下载
  • -P 下载到指定地址 (默认为.，即当前目录)
  • -t 最大尝试次数 (默认为20次)
  • -c 断点续传
  • -p 下载页面内所有资源，包括图片、视频等
  • -r 递归下载，默认最大深度为5

ps

ps命令用于查看进程信息

  • -a 显示所有进程
  • -u 显示用户以及其他详细信息
  • -x 显示没有控制终端的进程？？？

进程状态

  • D uninterruptible sleep (usually IO) 不可中断
  • R running or runnable (on run queue) 正在运行的或者是在运行队列中等待的
  • S interruptible sleep (waiting for an event to complete) 中断
  • T stopped by job control signal 停止
  • t stopped by debugger during the tracing
  • W paging (not valid since the 2.6.xx kernel)
  • X dead (should never be seen)
  • Z defunct (“zombie”) process, terminated but not reaped by its parent 僵死

top

top命令用于动态监视进程的活动与系统负载信息

pidof

pidof命令用于查看一个正在运行的程序的进程ID(PID)

pidof [参数] [服务名称]

kill

kill命令用于终止指定PID的服务进程

kill [参数] [进程PID]

killall

killall用于终止某个指定名称的服务所对应的全部进程

killall [参数] [服务名称]

通常一个复杂的服务程序会有多个进程协同工作，因此想要同时终止该服务程序的所有进程，通过killall指令代替kill命令，就不需要逐个输入每个pid去终止

在系统终端执行一个命令想要立即终止，用ctrl + C (有时候ctrl + Z?),就可以立即终止该命令。

如果不想显示命令屏幕的信息，可以在执行命令的末尾加上一个&，这样命令就在系统后台执行

系统状态检测命令

ifconfig

ifconfig命令用于获取网卡配置与网络状态等信息

ifconfig [网络设备] [参数]

其中，inet后面的参数是ipv4的地址，ether参数后面的是网卡物理地址(MAC地址)，RX是接收数据包信息，TX是发送数据包信息

uname

uname命令用于查看系统信息，系统内核信息

系统内核名称、主机名、内核发行版本、节点名、系统时间、硬件名称、硬件平台、处理器类型、操作系统名称

uname [参数]

查看当前系统的详细信息，通过查看centos-release文件（redhat操作系统则查看redhat-release）

uptime

uptime用于查看系统负载信息

显示当前系统时间、系统已运行时间、启用终端数量（当前登录的用户数量）以及平均负载值

  • 平均负载值指系统在最近1分钟、5分钟和15分钟以内的压力情况

  • 负载值越低越好， 不要长期超过1。生产环境不要长期超过5

对于文件/var/run/utmp储存着当前登录的用户信息

  • 这个文件不是通常的文本文件，而是二进制文件

对于文件夹/proc储存着进程信息

free

free命令用于显示当前系统中内存的使用量

free [-h]

  • total 内存总量
  • used 已用量
  • free 空闲量
  • shared 进程共享的内存量
  • buff/cache 缓存的内存量
  • available 可用量

内存信息储存在/proc/meminfo文件中

who

who命令用于查看当前登入主机的用户终端信息

who [参数]

显示登入本机的用户的名称、终端设备、登录到系统的时间

last

last用于查看所有系统的登录记录

last [参数]

last记录根据于/var/log/wtmp

使用last命令可以查看本机的登录记录。但是，由于这些信息都是以日志文件的形式保存在系统中，因此黑客可以很容易地对内容进行篡改。千万不要单纯以该命令的输出信息而判断系统有无被恶意入侵！

lastb

lastb类似于last命令，只不过是查看所有尝试不正确登陆的记录

lastb取决于文件/var/log/btmp

last和lastb的记录都是从文件生成开始的

history

history命令用于显示历史执行过的命令

history [-c]

使用参数-c会清空所有的命令历史记录

使用!数字方式来重复执行某一次命令

可以通过修改文件/etc/profile的HISTSIZE变量值来记录命令的条数

历史命令保存在用户家目录的.bash_history文件中

sosreport

sosreport命令用于收集系统配置及架构信息并输出诊断文档，格式为sosreport

其中生成文件在/var/tmp中

同时还有校验码checksum

工作目录切换命令

工作目录指用户当前在系统中所处的位置

pwd

pwd命令显示用户当前所处的工作目录

cd

cd命令用于切换工作路径

cd [目录名称]

  • cd - 切换到上一次所处的目录
  • cd .. 切换到上级目录
  • cd ~切换到当前用户的家目录
  • cd ~用户名 切换到其他用户的家目录

ls

ls显示目录中的文件信息

ls [选项] [文件]

  • -a 参数用于查看所有文件（包括隐藏文件）
  • -l参数可以查看文件的属性、大小等详细信息

文本文件编辑命令

cat

cat命令用于查看纯文本文件（内容较少的）

cat [选项] [文件]

  • 参数-n显示行号
  • cat 文件1 > 文件2 将文件1的内容覆盖到文件2
  • cat -n 文件1 > 文件2 将文件1的内容添加行号后覆盖到文件2
  • cat /dev/null > hello.txt：清空hello.txt
  • 向文本中添加信息

more

more命令用于查看纯文本文件（内容较多）

more [选项] 文件

使用空格或回车向下翻页

head

查看纯文本文件的前N行

head [选项] [文件]

  • head -n 数字 文件

tail

查看纯文本文件的后N行（类似head）

查看持续刷新的文件内容

  • tail -n 数字 文件名 :查看后N行
  • tail -f 文件名 ：持续查看刷新的文件

tr

tr命令用于替换纯文本文件中的字符

tr [原始字符] [目标字符]

wc

wc命令用于统计指定文本的行数，字数，字节数

wc [参数] 文本

  • -c只显示字节数
  • -l只显示行数
  • -w只显示单词数

stat

stat命令用于查看文件的具体储存信息和时间信息

cut

cut命令能够按列提取文本

cut [参数] 文本

  • -d后加分隔符。例如-d:
  • -f代表fields，即后面加需要查看被分割的第几部分

diff

diff命令用于查看多个文件内容的差异

文件目录管理命令

touch

touch命令可以用来创建空白文件，以及设置文件的时间（修改时间（modification time），文件读取时间（access time））

mkdir

mkdir用于创建空白目录，加参数-p用于创建出嵌套关系的文件目录

cp

cp用于复制文件或者目录

cp [选项] 源文件 目标文件

  • 如果目标文件是目录，则将源文件复制到该目录
  • 如果目标文件也是普通文件，则覆盖它
  • 如果目标文件不存在，则执行正常复制操作
  • -p，保留文件的属性
  • -d，若文件对象为链接文件，则保留该链接文件的属性
  • -r，递归持续复制（用于目录）
  • -i，若目标文件存在，则询问是否覆盖
  • -a，相当于-pdr

mv

mv用于剪切文件或者文件重命名

mv [选项] 源文件 [目标路径|重命名后的文件名]

源文件会被删除

rm

rm用于删除文件或目录

rm [选项] 文件

  • -r递归删除，用于删除目录
  • -f强制删除

dd

dd命令用于复制文件或者转化并格式化文件

dd [选项]

/dev/zero的设备文件不占用系统储存空间，但是可以提供无穷的数据

  • if=FILE 从FILE中读取数据而不是从标准输入读取
  • of=FILE输出到FILE中，而不是标准输出
  • by=BYTES一次性读写BYTES大小的数据
  • count=N复制N块输入块

同时dd命令可以制作光盘镜像文件iso

file

file命令用于查看文件类型

file 文件名

打包压缩与搜索命令

tar

tar命令用于对文件的打包压缩或解压

tar [选项] [文件] ….

  • -c：创建压缩文件
  • -x：解压文件
  • -t：查看压缩包有哪些文件
  • -z：用gzip压缩或解压文件
  • -j：用bzip2压缩或解压文件
  • -v：显示压缩或解压过程
  • -f： 目标文件名
  • -p：保留原始的权限和属性
  • -P：使用绝对路径来压缩
  • -C：指定解压到的目录
  • -c和-x不能同时使用
  • -f必须放在参数的最后一位

grep

用于在文本中执行关键词搜索

grep [选项] patterns [文件]

  • -n显示行号
  • -v反向选择，即不包含关键词patterns的行
  • -i忽略大小写
  • -c仅显示找到的行数
  • -b将可执行文件bianry当作文本文件text搜索

find

find命令用于按照指定条件查找文件

find [查找路径] 寻找条件 操作

find [-H] [-L] [-P] [-Olevel] [-D debugopts] [path…] [expression]

搜索可以根据文件不同的特性：文件名，大小，修改时间，权限等

默认的expression是-print

参数太多了，俺直接复制了
